# 🩺 Pranalyse  
### *AI-Powered Posture, Movement & Wellness Intelligence*

> **Pranalyse** is a smart health-tech platform designed to fix what generic fitness apps ignore i.e  **real-time feedback, posture correction, injury prevention, and personalized recovery**.

Millions of people, including working professionals, students, athletes, and home-based individuals, follow **one-size-fits-all** fitness, yoga, and diet advice.  
The result? 
❌ Poor movement, ❌ chronic pain, ❌ slow progress, and ❌ eventual drop-off.

Pranalyse changes that !!

## ⚙️ Setup & Installation

Follow these steps to run the project locally:

### 🔹 Prerequisites
Make sure you have the following installed:
- **Node.js** (v18+ recommended)
- **npm**
- **Flutter SDK** (latest stable)
- **Android Studio / VS Code** (with Flutter & Dart plugins)
- **Chrome / Android Emulator** (for Flutter app)

---

### 🔹 Project Setup

```bash
# 1. Clone the repository
git clone [REPO-LINK](https://github.com/ananya990/Pranalyse_)

# 2. Navigate to the project folder
cd pranalyse_web

# 3. Install web dependencies
npm install

# 4. Start the web development server
npm run dev

```

## 🧠 Tech Stack

### 🎨 Frontend
- **React.js** – Modern, component-driven web interface  
- **Tailwind CSS** – Utility-first styling for fast, responsive UI

### 📱 Mobile
- **Flutter** – Cross-platform mobile app development (Android & iOS)

### 🤖 AI & Computer Vision
- **MediaPipe** – Real-time posture and pose detection

### ☁️ Backend & Cloud
- **Firebase** – Authentication, backend services, and app integration  
- **Firestore** – Real-time NoSQL database  
- **Google Cloud Platform (GCP)** – Scalable cloud infrastructure and deployment
